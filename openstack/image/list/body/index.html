    (5) GET /v2/images¶

    openstack → glance-api

    API 참조 - List images

    GET /v2/images

    GET /v2/images API를 이용하여 이미지 목록 조회를 요청한다.
    query-string에 원하는 query-filters를 추가하여 조회 조건을 지정할 수 있다.
    자세한 검색 조건은 위의 List Images API를 참고한다.
